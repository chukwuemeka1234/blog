# Blog App

> The is a simple rails blog app that lets you navigate between users and their posts

## Built With

- Ruby on Rails
- Ruby

## Getting Started

To get a local copy up and running follow;

`git clone https://github.com/chukwuemeka1234/blog.git`

`cd blog`

Run the server with:

`rails s`

## Prerequisites

- Ruby
- Ruby on Rails
- PostgreSQL

## Install

- Ruby
- Ruby on Rails(as a gem)
- Postgresql database
